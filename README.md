### Setup

Clone the project

```sh
$ git clone https://github.com/pcavendao/react_ecommerce/
$ cd react_ecommerce

```

Install dependencies

```sh
$ npm ci
```

Start the app in development mode

```sh
$ npm start
```

Build app for production

```sh
$ npm build
```
